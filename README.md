# Dashboards
Dashboards do Power BI

Resumo:
Dashboards que criei, incluindo de serviço de Streaming (PrimeVideo), de evolução corporal (com dados de bioimpedâncias) e um currículo próprio em formato de Dashboard

Ferramentas utilizadas:
- Power BI (criação dos dashboards)
- Power Query (ETL e transformação de dados)
- DAX (criação de métricas e KPIs)
- Excel / CSV (fontes de dados)
- Modelagem de dados (relacionamentos entre tabelas)
- Git/GitHub (versionamento e portfólio)
- Figma/Canva (layout)

Aprendizados:
- Construção de dashboards orientados a decisão
- Aplicação de storytelling com dados
- Criação de KPIs relevantes para negócio
- Uso de DAX para métricas dinâmicas
- Processo de ETL com Power Query
- Organização e modelagem de dados (star schema básico)
- Design de dashboards (hierarquia visual, cores, clareza)
- Tradução de dados em insights acionáveis
- Pensamento analítico voltado para negócio
